A handy set of effects for use with [Hyperapp](https://github.com/jorgebucaran/hyperapp).

## Installation

<pre>
npm i <a href=https://www.npmjs.com/package/hyperapp-fx>hyperapp-fx</a>
</pre>

Then with a module bundler like [Rollup](https://rollupjs.org) or [Webpack](https://webpack.js.org), use as you would anything else.

```js
import { Http } from "hyperapp-fx"
```

If you don't want to set up a build environment, you can download Hyperapp FX from a CDN like [unpkg.com](https://unpkg.com/hyperapp-fx) and it will be globally available through the <samp>window.hyperappFx</samp> object. We support all ES5-compliant browsers, including Internet Explorer 10 and above.

```html
<script src="https://unpkg.com/hyperapp-fx"></script>
```
